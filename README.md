# Telegram Bot yigres

`$ npm run dev`

#### Telega-bot server start (Example)
`$ ssh root@94.26.237.173`  
`root@telega-bot:~# cd bot-yigres`  
`root@telega-bot:~/bot-yigres# pm2 start index.js`  
#### Telega-bot server start 
`root@telega-bot:~/bot-yigres# pm2 stop index.js`
